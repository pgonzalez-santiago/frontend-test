# Github Logo

----
## Props

* **height**             Number
* **style**              Object
* **width**              Number


----
## Usage

```html
  <GithubLogo
    width={40}
    height={40}
    style={{
      color:   'red',
      display: 'block',
      fill:    'currentColor',
      margin:  '0 auto',
    }}/>
```
