# Slider Item

----
## Props

* **onClick**        function: Callback function called when item is clicked.
* **name**              string
* **to**                string: Route link
* **stars**             number
* **watchers**          number


----
## Usage

```html
  <SliderItem
    onClick={() => null}
    name={'React'}
    to={'/react'}
    stars={1523}
    watchers={786}
  />
```
